# ShoeHaven: A Simple E-commerce Website

## Project Description

ShoeHaven is a simple, static e-commerce website designed to showcase a variety of footwear for men, women, and children. Built primarily with HTML and CSS, this project demonstrates fundamental web development concepts for creating a multi-page online store experience. It features product categories, an 'About Us' section detailing the brand's vision and team, a 'Contact Us' page, and a user login interface.

## Features

*   **Product Categories:** Dedicated sections for Men, Women, and Kids footwear.
*   **Interactive Homepage:** Includes a video showcase and featured product categories.
*   **About Us Page:** Provides detailed information about the brand, its milestones, team, customer testimonials, vision, mission, and social commitment.
*   **Contact Us Page:** Offers contact details and social media links.
*   **Login Page:** A basic login interface.
*   **Responsive Design:** (Implied by `meta name="viewport"` in HTML files, though not explicitly tested for full responsiveness, it's a common practice for modern web pages).

## Technologies Used

*   **HTML5:** For structuring the web content.
*   **CSS3:** For styling and layout.

## Setup and Installation

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Astik01/Simple-e-commerce-website.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Simple-e-commerce-website
    ```
3.  **Open `index.html`:**
    Simply open the `index.html` file in your preferred web browser to view the website.

## Usage

Navigate through the website using the top navigation bar. Explore different product categories, read about ShoeHaven's story, or get in touch via the contact page. The website is designed for straightforward browsing.
